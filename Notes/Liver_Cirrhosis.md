# Liver cirrhosis (肝硬化)

[TOC]

## 病因：
- 胆汁淤积
- 循环障碍
- 寄生虫感染
- 遗传和代谢

## 临床表现：
- 代偿期
	腹部不适、食欲减退、乏力、消化不良、腹泻等
- 失代偿期
	- 肝功能减退
		1. 消化不良
		2. 营养不良
		3. 黄疸
		4. 出血和贫血
		5. 内分泌失调
	- 门静脉高压
		1. 侧支循环形成
			- 食管胃底静脉曲张(EGV)
			- 腹壁静脉曲张
			- 痔静脉曲张
			- 腹膜后吻合支曲张
			- 脾肾分流
		2. 脾功能亢进及脾大
		3. 腹腔积液

## 并发症：
- 消化道出血
	- 食管胃底静脉曲张出血(EGVB)
	- 消化性溃疡
	- 门静脉高压性胃肠病
- 胆石症
- 感染
	- 自发性细菌性腹膜炎
	- 胆道感染
	- 肺部、肠道及尿路感染
- 肝性脑病(发病机制)
	- 氨中毒
	- 假性神经递质
	- 色氨酸
	- 锰离子
- 门静脉血栓或海绵样变
- 电解质和酸碱平衡紊乱
- 肝肾综合征(hepatorenal syndrome)
- 肝肺综合征(hepatopulmonary syndrome)
- 原发性肝癌

## 治疗：
- 保护和改善肝功能
	- 去除或减轻病因
	- 慎用损伤肝脏的药物
	- 维护肠内营养
	- 保护肝细胞
- 门静脉高压及其并发症治疗
	- 腹腔积液
		1. 限制钠、水摄入
		2. 利尿
		3. 经颈静脉肝内门腔分流术(TIPS)
		4. 排放腹腔积液加输注清蛋白
		5. 自发性细菌性腹膜炎：选用肝毒性小、针对革兰阴性杆菌兼顾革兰阳性杆菌的抗生素，如喹诺酮类
	- EGVB的治疗及预防
		1. 一般急救措施、积极补充血容量
		2. 止血措施
			* 药物：今早给予收缩内脏血管药物如生长抑素
			* 内镜治疗：出血量为中等以下，应紧急采用内镜结扎治疗(endoscopic variceal ligation, EVL)(不能降低门静脉高压)
			* TIPS：对急性大出血的止血率达到95%
			* 气囊压迫止血：暂时使用，为后续有效止血措施起“桥梁”作用
			* 一级预防：针对已有食管胃底静脉曲张，但尚未出血者
		3. 二级预防：对已发生过EGVB的病人，预防其再出血
	- 肝性脑病(HE)
		1. 及早去除HE发作的诱因
			* 纠正电解质和酸碱平衡紊乱
			* 预防和控制感染
			* 改善肠内微生态，减少肠内氮源性物质的生成与吸收
				- 止血和清除肠道积血：上消化道出血是HE的重要诱因之一
				- 防治便秘：可给予果乳糖
				- 口服抗生素：抑制肠道产尿素酶的细菌，减少氨的生成，如甲硝锉
			* 慎用镇静药及损伤肝功能的药物：镇静、催眠、镇痛药及麻醉剂可诱发HE
		2. 营养支持治疗 尽可能保证热能供应，避免低血糖；补充维生素
		3. 促进体内氨的代谢
		4. 调节神经递质
		5. 阻断门-体分流
	- 其他并发症治疗
		1. 胆石症
		2. 感染
		3. 门静脉血栓
		4. 肝硬化低钠血症
		5. 肝肾综合征
		6. 肝肺综合征
		7. 脾功能亢进
	- 手术
	- 病人教育
		1. 休息 不宜进行重体力活动及高强度体育锻炼
		2. 酒精及药物 严格禁酒
		3. 对已有食管胃底静脉曲张者，进食不宜过快、过多，食物不宜过于辛辣和粗糙
		4. 食物应以易消化、产期少的粮食为主
		5. 避免感染
		6. 了解肝硬化的病因
		7. 有轻微肝性脑病病人反应力较低，不宜驾车及高空作业
		8. 乙肝及丙肝病人可以与家人、朋友共餐。应避免血液途径的传染
